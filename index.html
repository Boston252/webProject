Els nostres Productes

Sistema d'Il·luminació Intel·ligent

  • Sensor de llum: Ajusta la il·luminació segons la quantitat de llum natural disponible.
  • Sensor de moviment: Encén i apaga les llums automàticament quan detecta moviment.
  • Control remot: Gestiona la il·luminació des del teu dispositiu mòbil.
  • LEDs d'alta eficiència: Proporcionen una il·luminació brillant amb un baix consum energètic.

Sistema de Seguretat Domòtica

  • Càmeres de seguretat: Monitorització en alta definició amb visió nocturna.
  • Sensor de moviment PIR: Detecta intrusos i envia alertes immediates al teu dispositiu mòbil.
  • Alarma intel·ligent: S'activa automàticament en detectar una intrusió.
  • Accés remot: Controla i visualitza les càmeres des del teu smartphone.

Control de Proximitat Intel·ligent

  • Sensor de proximitat: Detecta la presència de persones i ajusta el color en conseqüència.
  • Sensor de llum: Ajusta el color del led RGB segons quant t'aproximis.
  • Control remot: Gestiona el sensor i el led RGB del teu dispositiu mòbil.
  • Mode estalvi d'energia: Optimitza l'ús d'energia per reduir els costos.

Sistema de Persianes y Garatge

  • Sensor de proximitat: Detecta la presència de persones i para la porta per evitar accidents.
  • Control remot: Permet obrir i tancar les persianes i el garatge des del teu dispositiu mòbil.
  • Motor d'alta potència: Assegura un moviment suau i eficient de les persianes i la porta del garatge.
